# Classification Model Evaluation

## Objective

The objective of classification evaluation is to determine how accurately a machine learning model predicts categorical outcomes.

---

## Topics Covered

- Train-Test Split
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- KNN
- Support Vector Machine (SVM)

---

## Evaluation Metrics

### Accuracy

Measures the percentage of correct predictions.

Formula

Accuracy = Correct Predictions / Total Predictions

---

### Precision

Measures how many predicted positive cases are actually positive.

Used when False Positives are expensive.

Example:

Spam Email Detection

---

### Recall

Measures how many actual positive cases were correctly identified.

Used when False Negatives are dangerous.

Example:

Cancer Detection

---

### F1 Score

Balances Precision and Recall.

Useful when both False Positives and False Negatives matter.

---

## Real-world Example

Breast Cancer Classification

Models Used

- KNN
- SVM

---

## Key Learning

Accuracy alone is not enough.

Medical problems require high Recall.

Recommendation systems require high Precision.
