# Regression Model Evaluation

## Objective

Evaluate machine learning models that predict continuous numerical values.

---

## Metrics

### MAE

Average absolute error.

Easy to understand.

---

### MSE

Average squared error.

Large mistakes receive higher penalties.

---

### RMSE

Square root of MSE.

Same unit as the target variable.

Easy to interpret.

---

### R² Score

Measures how well the model explains the data.

Range

0 → Poor

1 → Perfect

---

## Model Used

Random Forest Regressor

---

## Dataset

California Housing Dataset

---

## Key Learning

Lower MAE, MSE, and RMSE are better.

Higher R² Score is better.

Residual plots help detect model errors.

Feature Importance identifies the most influential variables.
