# Unsupervised Learning Evaluation

## Objective

Evaluate clustering performance without labeled data.

---

## Topics

- K-Means
- Inertia
- Silhouette Score
- Davies-Bouldin Index
- Elbow Method
- Cluster Stability

---

## Dataset

Synthetic Blob Dataset

---

## Evaluation Metrics

### Inertia

Lower is better.

Measures compactness.

---

### Silhouette Score

Range

-1 to 1

Higher is better.

---

### Davies-Bouldin Index

Lower is better.

Measures cluster similarity.

---

## Key Learning

No single metric should be used alone.

Always compare multiple metrics before choosing K.
